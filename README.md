A backend-driven Recipe Management application built using Django REST Framework, enhanced with Generative AI to simplify recipe instructions for beginners. The project demonstrates clean REST API design, data handling, and responsible integration of AI services.

Project Overview

Smart Recipe Explorer allows users to:

Create and manage recipes

Search and filter recipes

Use AI to simplify complex cooking instructions into easy-to-follow steps

The application focuses on backend architecture, API design, and AI integration rather than UI complexity.
